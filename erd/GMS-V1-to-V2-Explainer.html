Change 1: SVT Archive Field Naming

DB-202 (Database Static Value Tables)
"All SVT tables must have the fields listed in the table below..."
Including: ArchivedByID (int, optional)

What Changed

All 9 SVT tables had ArchivedByAccountId renamed to ArchivedByID.

Why: DB-202 explicitly specifies ArchivedByID for SVT tables, while DB-113 specifies ArchivedByAccountId for regular tables. These are different standards.

Change 2: Foreign Key Naming Corrections

DB-105 (Foreign Key Naming)
"Foreign keys are named 'Id' prefixed with the name of the foreign key's parent table."
DB-107 (FK Shortening Exception)
"When a foreign key's parent table name includes the primary table's name, the foreign key name can be shortened to avoid redundancy, assuming it does not create ambiguity."

What Changed

3 FK columns were renamed because DB-107's shortening exception did NOT apply to them.

Change 3: AuditLog Missing Audit Fields
